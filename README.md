# Hands-On Machine Learning Practice

This repository is my personal practice space for working through examples and exercises from *Hands-On Machine Learning*.

## Purpose

- Practice core machine learning concepts from the book
- Reproduce examples using real datasets
- Keep chapter-wise notebooks and experiments in one place



## Workflow

1. Open notebooks and run cells step by step
2. Explore data, visualize distributions, and test preprocessing ideas
3. Keep notes and variations directly in notebooks

## Environment

This project is run in a Python virtual environment (`.venv`).

Typical packages used:

- `numpy`
- `pandas`
- `matplotlib`
- `scikit-learn`

## How to Run

1. Activate your virtual environment
2. Open the notebook (`.ipynb`) in VS Code or Jupyter
3. Run cells in order to reproduce results

## Notes

- This repo is for learning and practice, not a production ML project
- Code may include exploratory steps and intermediate outputs
